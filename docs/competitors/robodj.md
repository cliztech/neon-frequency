# RoboDJ (AI Add-On) - Feature Analysis

## Source URLs
- https://studio.radiodjdude.com/robodj/
- https://www.youtube.com/watch?v=GkbWTqGP8K8

## Core System Concept
- **Type:** AI voice tracking automation layer (add-on).
- **Function:** Connects to existing automation systems (PlayIt Live, RadioDJ, StationPlaylist, NextKast, etc.) to inject live-sounding AI voice tracks.

## Pricing
- Buy-out pricing noted at $820 lifetime or $400/year with no contracts (from provided notes).

## Manuals & Resources
- Primary documentation is a video tutorial library provided after purchase; no standalone PDF was identified in the provided notes.

## Feature List (V2 & V3 Sneak Peek)
- **AI Personalities:** Supports ElevenLabs and Google Gemini voices (up to 16 voices) with multi-voice scripts for "morning show" banter.
- **Content Generation:** Dual marker mode (back/front announce), playlist-aware trivia/promos, smart scheduler markers, banned phrase tool, and dynamic prompt variables (time of day, station name, tags).
- **Real-Time Info:** Weather integration via National Weather Service or Weather API with music bed mixing.
- **Audio Processing:** Loudness matching plus a silence remover to tighten AI pacing.
- **Platform:** Windows 8/10/11 only.

## UI/UX Notes
- **Interface:** Module-based settings (weather, voices, prompts).
- **Workflow:** Background "set it and forget it" operation that monitors the automation log/output.
- **Customization:** Prompt Doctor tool to refine AI scripts before airing.
