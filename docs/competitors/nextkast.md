## Core System Architecture
- **All-in-One Design:** Integrates Automation, Scheduler, Traffic/Billing, Remote Access, Streaming, and Audio Processing in a single installer.
- **File-Based Metadata:** Stores cue points, rotations, and metadata in the file system for resilience and easy backups.
- **Efficiency:** Low CPU/memory footprint; designed for standard Windows 10/11 PCs and virtual servers.
- **Encoders:** Built-in encoders for Shoutcast, Icecast, and Live365 with dual output support (soundcard + stream).

## Key Features
- **Professional Music Scheduling:** Built-in clocks, rotations, and rules with Audience Pleaser (tune-in/out stats), NextBalance smoothing, and Repick Category fillers for top-of-hour timing.
- **nextAI Virtual Host:** AI intros/outros, energy matching, voice cloning, and gap-filling for off-hours.
- **Remote Operations (StudioLink & GoLive):** Web-based remote voice tracking with permissions, plus WebRTC multi-party live hosting.
- **nextSync (Multi-Market):** Pull-based syndication with central content distribution and local ad/imaging support.
- **Traffic & Billing:** Integrated log merging, invoicing, reconciliation, and rotation integration without templates.
- **Audio Processing:** Built-in AGC, 10-band EQ, multi-stage compression, and Sound Expander.
- **Interoperability:** Livewire+ partner support, MusicMaster Nexus integration, and RDS/TuneIn metadata push.

## UI/UX Notes
- **Philosophy:** Clean interface with drag-and-drop workflow.
- **Playout Screen:** Configurable single-screen layout with direct drag-in playout.
- **Visuals:** Dark mode dominant with color-coded categories.

## Feature list (from manual)
- Category-based music library with drag-and-drop rotation building.
- Rotation rules including top-of-hour forcing and auto-start on app launch.
- Playlist builder with saved playlists alongside reusable rotations.
- Scheduler for rotations, playlists, or line-in events by hour or minute.
- Track metadata flags for sweepers, intros, outros, and next-start cues.
- Overlay sweepers that play over transitions.
- Manual, Live Assist, and Automated operation modes (with semi-automation while browsing).
- Sound FX/Hot Button cart wall (up to 100 hot buttons).
- Voice tracking inside the playlist builder.
- Broadcast settings for Shoutcast/Icecast (sound card selection).
- Simulcast another stream with playlist generation from an external source.
- Microphone/line-in mixing and mic FX toggles.
- External DSP plugin chain support (e.g., Stereo Tool, Breakaway).
- Remote voice tracking and library sharing between machines.
- Skype/VOIP call integration.
- Shortcut keys for main UI and voice tracking.
- Backup/restore utility for library + configuration.
